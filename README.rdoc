== Recao

Recao is a yelp clone dedicated to vegetarians and vegans. Find, add and review your favorite vegetarian-friendly restaurants in Puerto Rico.

== Key Features
- User dashboards, with all of the user's comments, reviews and listed places.
- Mobile-First responsive design, using Bootstrap
- Pagination using Kaminari gem

Find it live here: http://nomster-eric-crescioni.herokuapp.com/
